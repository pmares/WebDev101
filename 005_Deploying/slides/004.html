Image Processing

Image Magick is a software suite to create, edit, compose, or convert bitmap images.

Original image, upside down, large file size

Resize and Rotate Example

convert Merlin.jpg \ -resize 25% \ -auto-orient Merlin.small.jpg

Smaller image is only 4% of original size (reduced from 5.6MB to 0.21MB)
