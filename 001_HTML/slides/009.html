Basic Tags

  • <!DOCTYPE>: declaration must be the very first thing in your HTML document
  • <html>: tag is the container for all other HTML elements (except for the tag).
  • <head>: element can include a title for the document, scripts, styles, meta information, and more.
    The following elements can go inside the <head> element:
    • <title> tag is required in all HTML documents and it defines the title of the document.
    • <style> tag is used to define style (CSS) information for an HTML document.
    • <base> tag specifies the base URL/target for all relative URLs in a document.
    • <link> tag defines a link between a document and an external resource.
    • <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.
    • <script> tag is used to define a client-side script (JavaScript).
    • <noscript> tag defines an alternate content for users that have disabled scripts in their browser or have a browser that doesn't support script.
  • <body> tag defines the document's body. This element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
  • <!--...-->: The comment tag is used to insert comments in the source code./li>
  • <p> tag defines a paragraph.
