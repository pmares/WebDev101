Nesting Tags

We have already seen in the case of formatting tags, such as having styled text and [links] within a paragraph. tags can be nested within each other. Some elements need to be nesting to work properly. 

<ol>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ol>

<ul>
  <li>Dog</li>
  <li>Goat</li>
  <li>Alpaca</li>
</ul>
  1. One
  2. Two
  3. Three
  • Dog
  • Goat
  • Alpaca
