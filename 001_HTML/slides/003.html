Brief History of the World Wide Web

  • 1989 - Tim Berners-Lee wrote proposal for the WWW based on Hypertext
  • 1990 - HTML / URI / HTTP
    • First web editor/browser
    • First web server
  • 1991 - First public web service
  • 1993 - Mosaic web browser released
  • 1994 - Netscape browser released
  • 1994 - Creation of W3C, the World Wide Web Consortium
  • 1995 - Internet Explorer released
    • Javascipt
    • Start of the browser Wars
  • 1997 HTML 3.2
  • 1999 HTML 4.01 standard
  • 2014 HTML 5 standard

See: Web Foundation History
