Transpilers

The current versions of Javascript are not widely supported on modern browsers. ( see JavaScript Compatibility ) In order to start using the features and syntax available in ES6 & 7, the use of transpilers is gaining in popularity.

A transpilers is a program that compiles new JavaScript syntax into an older, more supported version of the language.

In order to use a transpiler, you must use an external JavaScript file with a .js extension

Babel has support for the latest version of JavaScript, and this transpiler is supported within codepen.
