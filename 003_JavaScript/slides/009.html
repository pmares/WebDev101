Events

HTML events are "things" that happen to HTML elements.

When JavaScript is used in HTML pages, JavaScript can "react" on these events.

Examples include:
  • moving the mouse over an element
  • clicking on an element
  • changing text in a form
  • loading a page

demo.html
<form> <input id="demo-input" type="text" oninput="myFunction()"> </input> </form> <div id="demo-div"> </div>
demo.js
function myFunction() { var inputElement = document.getElementById("demo-input") var divElement = document.getElementById("demo-div"); divElement.innerHTML = inputElement.value; }
