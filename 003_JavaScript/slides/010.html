Context

What is this

The Javascript object named "this" is the owner of the code that you are currently running. JavaScript code that is not tied to any other object, class, or element has the context of the browser's Window.

If you open the console in your browser, "this" will return a reference to the Window object

When an event "fires" the "this" is set to target element, the object that owns the event.

demo1.html
<button onclick="window.console.log(this)"> Click Me </button>

demo2.html
<button onclick="myFunction()"> Click Me </button>
demo2.html
function myFunction() { window.console.log(this); }

demo3.html
