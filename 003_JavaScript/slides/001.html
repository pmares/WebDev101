Introduction to Javascript

Alongside HTML and CSS, Javascript is one of the three core technologies of World Wide Web content production; the majority of websites employ it and it is supported by all modern Web browsers without needing additional plug-ins.

The 6th edition of Javascript, officially known as ECMAScript 2015, was finalized June 17, 2015

The 7th edition of JavaScript, ECMAScript 2016, was finalized in June 2016.
