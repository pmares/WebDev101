Selectors

You tell the browser to apply your CSS to some elements and not others with selectors.

element

div { font-family: Sans-serif; } <div> Generic Division </div>

#id

#my-text { font-align: center; } <div id="my-text"> My Text </div>

.class

.blue-box { width: 200px; height: 200px; background-color: light-blue; } <div class="blue-box"> Example Box </div>
