Fonts and Text

Default font

Changing the font on your web side is easy, but there is a small catch. Not all computers and browsers have the same fonts. In order to support the widest range of browers, the font-family property supports "fallback" font names.

Using the body element selector will change the font for the entire page

Sans Serif Font

body { font-family: "Lucida Sans Unicode", "Lucida Grande", sans-serif; }

Serif Font

body { font-family: "Palatino Linotype", "Book Antiqua", Palatino, serif; }

Monospaced Font

body { font-family: "Lucida Console", Monaco, monospace; }
