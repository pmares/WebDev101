Margins, Padding, and Borders

Margins are the space on the outsize of the element

Padding is the space on the inside of the element

Margin
Border
Padding
Content

You can specify the sizes of the properties in several ways

  • One value for all sides
    .padding-even { padding: 20px; }
    Even Padding
  • Two values, [top & bottom] [left & right]
    .padding-wide { padding: 10px 60px; }
    Wide Padding
  • Or specify each side individually
    .odd-padding { padding: 0 20px 40px 60px; }
    Uneven Padding
