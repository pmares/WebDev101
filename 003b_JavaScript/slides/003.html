typeof operator

The typeof operator returns a string indicating the type of the unevaluated operand.

typeof "Testing" // "string" typeof 1234.543 // "number" typeof true // "boolean" typeof { x: 1 } // "object" typeof function(){} // "function" typeof undefined // "undefined" typeof x // "undefined" (Assuming you have not declared x) typeof Symbol('test') // "symbol" typeof null // "object"
