Conditionals

The if statement executes a statement if a specified condition is true.

if (condition1) statement1

If the condition is false, another statement can be executed.

if (condition1) statement1 else statement2

Multiple if...else statements can be nested to create an else if clause.

if (condition1) statement1 else if (condition2) statement2 else if (condition3) { statement3; statement4; statement5; }
