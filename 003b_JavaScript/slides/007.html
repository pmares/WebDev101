Functions...

parameters

Functions can optionally take arguments (aka parameters) -- values you pass in.

example.js
function myFunction(name) { console.log("Hello " + name + " how are you today?"); }
console
> myFunction('Phil') Hello Phil, how are you today?

return values

functions can also optionally return a value

example.js
function addTwo(someNumber) { return someNumber + 2; }
console
> var x = addTwo(5); 7
